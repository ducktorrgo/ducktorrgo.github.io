Search Torrminatorr directly by default in a Browser!

    For Firefox

    1. Click the green + in the Search Box
    2. Add Torrminatorr to your search options
    3. Set it as Default in Preferences / Search

    For Chrome

    1. Open Settings / Search Engine
    2. Click Manage / Add to Fullfill with:

      3. Torrminatorr
      TRR
      https://forum.torrminatorr.com/search.php?keywords=%s&sf=titleonly&sr=topics&submit=Search

    3. Make the engine Default by ⋮
