TorrTorrGo

Search Torrminatorr with any words by default in a Browser!

    For Firefox

    1. Click the green + in the Search Box
    2. Add TorrTorrGo to your search options
    3. Set it as Default in Preferences / Search

    For Chrome

    1. Open Settings / Search Engine
    2. Click Manage / Add. Fullfill with:

      3. TorrTorrGo
